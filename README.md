# TryHackMe-Defensive-security-intro
Registro prático dos conhecimentos obtidos neste laboratório da plataforma TryHackMe

![Conclusão do Laboratório](https://raw.githubusercontent.com/Vitor-Oliveira-cyber/TryHackMe-Defensive-security-intro/refs/heads/main/ConclusaoDoLab.png)


## Conteúdos abordados
- Diferença entre eventos, alertas e incidentes;
- Tentativas de login mal sucedidas como indicador de ataque;
- Importância do monitoramento contínuo.

  ## O que aprendi
 - Nem todo evento é um incidente;
 - Padrões repetitivos podem indicar atividades maliciosas;
 - A importância de analisar o contexto antes de escalar um alerta.

    ## Conclusão
    O Laboratório apresentou conceitos fundamentais de segurança defensiva e o papel do analista SOC na identificação e resposta a possíveis incidentes.
